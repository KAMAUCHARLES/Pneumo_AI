========================================
PneumoAI – AI-Powered Pneumonia Detection
========================================

How to run:
1. Extract the ZIP folder to a location of your choice.
2. Double-click "run_pneumoai.bat".
3. Wait while the setup completes (first run only).
4. Your browser will open with the app.

Requirements:
- Windows 7/8/10/11 (64-bit)
- Internet connection for first run (to download Python and packages)
- After first run, you can use it offline.

Troubleshooting:
- If the batch file fails, ensure your antivirus is not blocking the download.
- You can manually download Python embeddable from python.org and extract to a folder named "python_portable" inside this folder.

For any issues, contact the team.
